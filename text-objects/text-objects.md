# TEXT OBJECTS

* define regions of text by structure

!SLIDE bullets
# "quotes"
* `a"` - a quoted string
* `i"` - inner quoted string

!SLIDE bullets
# (brackets)
* `ab` - a block
* `ib` - inner block

!SLIDE bullets
# {braces}
* `aB` - a Block
* `iB` - inner Block

!SLIDE bullets

# `<xml>`tags`</xml>`

* `at` - a tag block
* `it` - inner tag block
